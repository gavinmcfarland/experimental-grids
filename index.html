Experimenting with bullet proof grids

The following is are examples using shadow DOM to wrap each child element of a container with wrapper divs. This then provides the offardance to create layouts which are flexible in a lot of scenarios.

  1. Negates need to include extra divs when writing your markup
  2. Use both pixels and percentages for guttering (margins)
  3. No need to append extra class to last element of row
  4. Complete freedom in styling child element, eg use borders, padding etc
  5. Good for creating responsive layouts
  6. Very very flexible and will accomodate a lot of layouts
